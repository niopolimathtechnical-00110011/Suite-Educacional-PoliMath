# 🧮 Suíte PolyMath Educacional

Suite educacional para demonstrar funcionamento de programas em Windows nos modos texto e gráfico.

- **Nome:** Enio Alves Borges
- **GitHub:** @niopolimathtechnical-00110011
- **Projeto:** Suíte PolyMath Educacional

## 📦 Aplicativos Inclusos

1. **Calculadora** (CMD e GUI)
   - Operações básicas: +, -, *, /
   - Interface simples e intuitiva

2. **Cadastro de Alunos** (CMD e GUI)
   - CRUD completo (Create, Read, Update, Delete)
   - Persistência em arquivos
   - Suporte a criptografia de dados

## 📁 Estrutura de Dados

- `db_alunos.polymath` - Dados não criptografados
- `db_alunos.polymath_crypt` - Dados criptografados
- `key.key` - Chave de criptografia (gerada automaticamente)
